## Introduction 📌
Web now a days contains all the information required by a person. Web scrapper is a application that can help in scrapping the web and finding the usefull information form it. This project is made with the use of Black Coffer Blogs with their permission to use these links.
## Technology Stack 🏁

* [Python](https://www.python.org/)
* [Pycharm](https://www.jetbrains.com/pycharm/)
* [Jupyter Notebook](https://jupyter.org)
## Dependencies used:
  
* [Requests](https://pypi.org/project/requests/)
* [NLTK](https://pypi.org/project/nltk/)
* [BeautifulSoup](https://pypi.org/project/bs4/)
* [Pandas](https://pypi.org/project/pandas2/)
* [openpyxl](https://pypi.org/project/openpyxl/)
